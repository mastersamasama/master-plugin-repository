# example-skill

A reference skill that does nothing useful — its purpose is to demonstrate the file structure of a SKILL.md inside a multi-component plugin.

## When invoked

Tell the user: "This is example-skill from multi-component-plugin. The same plugin also ships /multi-component-plugin:example-command, an agent named example-agent, and a PreToolUse hook."

## Purpose

This skill is purely educational. Real plugins should have skills that do real work.
