# minimal-plugin (example)

The smallest possible Claude Code plugin: one `plugin.json` + one `SKILL.md`.

```
minimal-plugin/
├── .claude-plugin/
│   └── plugin.json
├── skills/
│   └── hello/
│       └── SKILL.md
└── README.md
```

Use this as a reference for the minimum file set every plugin must have. Copy and adapt for your own submission.
