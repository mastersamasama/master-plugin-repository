# x-social

Privacy social platform with ZK face verification, AI matchmaking, and privacy payments on OKX X Layer

> Submitted to the Claude Code plugin marketplace at `mastersamasama/master-plugin-repository`.

## Install

```
/plugin marketplace add mastersamasama/master-plugin-repository
/plugin install x-social@master-plugin-repository
/reload-plugins
```

## What's inside

- **Skills**: `deploy-contracts` (deploy X-Social smart contracts to X Layer), `privacy-pay` (Railgun ZK-SNARK privacy payments)
- **Commands**: `x-social-setup` (one-command local dev environment setup)
- **Agents**: `matchmaking-agent` (AI matchmaking with ZK verification and ERC-8004 reputation)
